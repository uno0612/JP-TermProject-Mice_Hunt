# JP-TermProject-Mice_Hunt
22300013 강운 (Un Kang), 22300835 김호훈(Hohun Kim), 22500373 Bibek Sunar

A mini game, Mice Hunt, has 10 randomly generated stage, each stage lasts for limited time.
You can choose difficulty easy, normal, hard.
You must catch all the mice to move into the next stage. As you move up a stage, mice count increases.
The Highscore would be # of caught mice.
When the time runs out before you catch all mice, game over.
If you complete all 10 stages, you win!
You can restart by pressing R when you are in Game Over or Win Screen.
